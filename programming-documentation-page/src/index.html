Object-Oriented Programming

Definition Classes Objects Methods Functions Variables Arrays

Definition

Object-Oriented Programming is an approach to problem solving. It has become the most widely used approach to software development

Classes

A class is a concept / a blueprint / a basic structure of an object. They're very useful in programming when we need several instances of an object because it avoids us the work of describing each object individually.

For example, a car is an instance of the class Vehicle.

Objects

An object is a component of a program that knows how to perform certain actions and how to interact with other elements of the program. Objects are the basic units of object-oriented programming.

To continue with our example: car = Vehicle.new

Methods

A method is a procedure that defines the behavior of the objects created from a class. We can think of them as actions that an object can perform

Functions

A function is a combination of instructions that need some input (arguments) and is meant to achieve some result.

The difference between a function and a method is that the functions is independent. That means that you can use it anywhere and it doesn't need an object to work.

Variables

A variable is a container that holds information that can be referenced and manipulated in a computer program.

Types of variables

  • Constants

    • They are used for storing data that never needs to change.

    MY_CONSTANT = "I am a constant and I'll never change"

  • Global variables

    • They are available throughout your entire app, overriding all scope boundaries.

    $global_variable = "I'm available through your app"

  • Class variables

    • They are accessible by instances of your class, as well as the class itself.

    @@brand = "Audi"

  • Instance variables

    • They are available throughout the current instance of the parent class.

    @var = 'I am available throughout the current instance of this class.'

  • Local variables

    • They are the most common variables you will come across and obey all scope boundaries.

    var = 'I must be passed around to cross scope boundaries.'

Arrays

An array is a data structure, which can store a fixed-size collection of elements of the same data type. An array is used to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type.
